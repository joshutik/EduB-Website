BLOCKLY

Blockly is simply a series of educational games that teaches programming. With Blockly, learning is self-paced and designed for people who have little or no knowledge of programming. By the end of these games, players are ready to use conventional text-based languages. Our Blockly course will enable users to create scripts and programmes by using visual blocks, even though they do not know any programming language. It includes everything you need for defining and rendering blocks in a drag-n-drop editor. Blockly provides a platform for our students to play games and learn the concept of programming simultaneously.

DATA SCIENCE

In recent times, Data Science has become one of the most
profitable and most sought-after courses. It encompasses preparing
data for analysis, including cleansing, aggregating, and manipulating
the data to perform advanced data analysis.

In our data analysis course, our students should:

1. Have the entire toolbox needed to become a data scientist.
2. Have acquired in-demand data science skills: Statistical analysis, Python programming with NumPy, pandas, matplotlib, and Seaborn, Advanced statistical analysis, Tableau, Machine Learning with stats models and scikit-learn, and Deep Learning with TensorFlow.
3. Have an understanding of the data science field, learn how to pre-process data, understand the mathematics behind Machine Learning (an absolute must which other courses don’t teach).
4. Start coding in Python, learn how to use it for statistical analysis, and perform linear and logistic regressions in Python.
5. Be able to carry out Cluster and Factor analysis, create Machine Learning algorithms in Python using NumPy, statsmodels, and scikit-learn.
6. Be able to apply skills to real-life business cases, use state-of-the-art deep learning frameworks such as Google’s TensorFlow, develop a business intuition while coding and solving tasks with big data, and unfold the power of deep neural networks.
7. Be able to improve Machine Learning algorithms by studying underfitting, overfitting, training, validation, n-fold cross-validation, testing, and how hyperparameters could improve performance.
